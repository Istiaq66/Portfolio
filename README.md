# Istiaq Ahmed — Portfolio

Personal portfolio site for **Istiaq Ahmed**, Flutter Developer & Software Engineer. Single-page static site with custom cursor, particle background, animated timeline, and project showcase.

🔗 **Live:** <https://istiaq66.github.io/Portfolio/>

## Sections

- **Home** — hero intro
- **Career** — timeline
- **Skills** — tech stack
- **Projects** — featured work
- **Contact** — get in touch
